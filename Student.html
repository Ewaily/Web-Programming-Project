Web Programming Project | Registration System

A Registration System for AAST is to be developed with a front-end Interface using HTML+CSS+JS and a back-end using PHP+MySQL

The website enables students to register courses and view their schedule.

Main Features of the Website:

  1. Admins can create Lecturers, Students and Courses.
  2. Lecturers can view students enrolled in courses and view their schedule.
  3. Students can register courses and view their schedule.

PART1: Build the website using HTML5 & CSS

[Deadline: 7th Week]

Your web site should contain at least the following pages:-

  1. A home page.
  2. A contact us page which contains a feedback form & contact info.
  3. About Page which contains info about the website & the services available.
  4. News Page which display the latest college news from the official facebook page.
  5. You can add any other pages you would think relevant to your site.

All the following regulations should be met:-

  1. Web pages should be HTML5 and should be validated successfully.
  2. Use CSS to apply styles and format, your CSS should be also valid.
  3. All pages should have appropriate titles.
  4. For all links and files, use relative path rather than fixed path.
  5. Use meta tags to include a series of keywords that describe your document.
  6. All pages should use consistent styles.
